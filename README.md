# Лабораторная работа №12: Автоматизированное тестирование с Selenium WebDriver

## Описание
Автотесты для GitHub, написанные на Python с использованием Selenium WebDriver.

## Тест-кейсы

| № | Название | Ожидаемый результат |
|---|----------|---------------------|
| TC-001 | Успешная авторизация | Пользователь переходит на главную страницу, отображается аватар |
| TC-002 | Неуспешная авторизация | Отображается сообщение об ошибке |
| TC-003 | Переход к репозиторию | Открывается страница репозитория china |

## Установка и запуск

```bash
# Установка Selenium
pip install selenium

# Запуск тестов
python github_autotests.py
